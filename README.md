### 软件介绍

Snow Shot 是一款功能强大的截图工具，集成了文字识别、翻译和 AI 对话等功能。它采用简洁优雅的设计理念，致力于为用户带来流畅高效的使用体验。

*注意：软件正在快速迭代中，Bug 和使用问题难以避免。且因为尚处测试阶段，非本地功能（如翻译和 AI 对话）的 API 服务可能日后不会兼容旧的客户端，如果你试图寻求一个稳定的工具软件，目前请谨慎选择当前软件。*
